
# VarStream Syntax Diagram

## Variable
<var> = <value>
Set <var> as <value>
Let <var> be <value>

## Condition
If <expr> then <action>
If <expr> then <action> else <action>

## List
Create a list of numbers <start> to <end>
Make a list called <name> containing <items>
Add item <value> to list <name>

## Dictionary
create dict <name> : key 'value', key 2
set_key <dict> <key> <value>

## Function
Define function name(args) as statements; return value
Call name with args

## Repeat
repeat <n> times <action>
