
# VarStream Official Documentation

## Overview
VarStream is a natural-language scripting environment designed for beginners.

## Capabilities
Supports:
- Variables
- Math
- Conditions
- Lists
- Dictionaries
- Inputs
- Functions
- Repeat loops

## Execution Flow
1. Normalize human text
2. Convert to canonical commands
3. Execute line-by-line
4. Display output for `show`, `return`, and inputs

## Example Script
x = 9
Check if x equals 9
If x equals 9 then show "Yes"

## System Requirements
Python 3.10 only

## Run a File
python varstream.py program.vs
